# Collective-Access-Site

Collective Access Website: Developing a SaaS platform to deploy and manage art collection software (Collective Access) via Kubernetes pods.

Context: Providing a dedicated SaaS platform for private and public art collectors.
Technical Vision: Creating a website where users can purchase subscriptions that trigger the automated deployment of dedicated Collective Access Kubernetes pods.
AI Integration: Implementing an AI-driven support chatbot to manage tickets and assist with software evolution.
